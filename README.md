## Theme
Quiz or Trivia Game - Interactive quiz application with 3 difficulty levels

## Features Implemented
1. **3 Difficulty Levels** (Easy, Medium, Hard)
2. **Timer Functionality** - 60 seconds per question
3. **Dynamic Content Loading** - Questions load based on selected level
4. **Score Tracking** - Real-time scoring and results page
5. **Responsive Design** - Works on all devices using Bootstrap
6. **Form Validation** - Contact form with validation
7. **Interactive UI** - Hover effects, animations, and transitions

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome Icons

## How to Run the Project
1. Download/Extract all files maintaining the folder structure
2. Open `index.html` in any modern web browser
3. Click "Play Now" to start the quiz
4. Select difficulty level and enjoy!
